# محوّل المكتبة الشاملة إلى مندلي · Shamela → Mendeley

أداة احترافية لتحويل قواعد بيانات **المكتبة الشاملة** — وأي قاعدة **SQLite** — إلى صِيَغ ببليوغرافية يقبلها **Mendeley** و**Zotero** و**Microsoft Word**، مع مولّد أنماط **CSL** عربي مبني على Chicago 17.

> **قراءة فقط بضمان فيزيائي:** كل اتصال بالقاعدة يُفتح بوضع `mode=ro` مع `PRAGMA query_only=ON`؛ يستحيل تعديل القاعدة المصدر أو الحذف منها.

---

## ✨ المزايا

- **تصدير متعدّد الصِيَغ:** BibTeX (`.bib`) · RIS (`.ris`) · **Word XML** (`.xml`) لاستيراد مصادر مايكروسوفت وورد.
- **لا-مخططية (Schema-Agnostic):** لا تفترض الأداة بنية جداول مسبقة؛ تستبطن القاعدة وتربط الأعمدة بالحقول الببليوغرافية تلقائياً أو يدوياً.
- **حفظ الأسماء العربية المركّبة كاملة:** أسماء مثل «عبد الله بن محمد» و«ابن خالويه» و«آل فلان» لا تتقطّع في أي صيغة (في Word XML تُوضع كوحدة واحدة في `<b:Last>`).
- **حفظ الأدوار العلمية:** المراجِع/المعلِّق/الراوي/الجامِع/المقدِّم/المشرف تُطوى في حقل الملاحظة فلا تُفقَد.
- **تنظيف مطبعي ذكي:** إزالة البادئات والآثار التحريرية ووسوم مثل `#مستنتج#` و`[...]`، مع خيارات قابلة للتهيئة.
- **وضع «بطاقة الشاملة» (betaka):** تفكيك النص الحرّ إلى حقول.
- **إزالة المكرّرات** بمفتاح العنوان + المؤلف.
- **واجهة رسومية عربية RTL** (PyQt6) + **واجهة سطر أوامر** للتشغيل بلا واجهة.
- **مرسم تدبر لأنماط CSL:** صفحة ويب تفاعلية متجاوبة تولّد ملف `tadabbur.csl` جاهزاً لمندلي، مع تدقيق صحّة حيّ.

## 📂 مكوّنات المشروع

| الملف | الوصف |
|------|-------|
| `bib_engine.py` | النواة المنطقية النقية (SQLite → BibTeX/RIS/Word XML). بلا اعتماد على واجهة. |
| `shamela_to_mendeley_gui.py` | الواجهة الرسومية (PyQt6) فوق النواة. |
| `tadabbur-csl-studio.html` | مولّد أنماط CSL لمندلي (ملف واحد، يعمل في المتصفّح بلا بناء). |
| `smart_dictionary.db` | قاعدة بيانات SQLite المصدر *(انظر تنبيه الحجم أدناه)*. |

## 🛠️ المتطلّبات

- **Python 3.9+**
- **PyQt6** (للواجهة الرسومية فقط؛ النواة وسطر الأوامر يعملان بمكتبات بايثون القياسية وحدها).
- متصفّح حديث (لمرسم CSL).

```bash
pip install PyQt6
```

## 🚀 التشغيل

### 1) الواجهة الرسومية

```bash
python shamela_to_mendeley_gui.py
```

ثم: حدّد ملف القاعدة ← اختر الجدول ← اضبط ربط الحقول (أو «تعرّف تلقائي») ← اختر الصيغة ← «بدء الاستخراج والتصدير».

### 2) سطر الأوامر (بلا واجهة)

```bash
python bib_engine.py \
  --db smart_dictionary.db \
  --table books \
  --out books_mendeley.bib \
  --format wordxml \
  --extra-roles
```

**الخيارات:**

| الخيار | المعنى |
|--------|--------|
| `--db` | مسار قاعدة البيانات (مطلوب). |
| `--table` | اسم الجدول المصدر (مطلوب). |
| `--out` | مسار ملف الإخراج (مطلوب). |
| `--format` | `bibtex` · `ris` · `wordxml` (الافتراضي `bibtex`). |
| `--title-col` / `--author-col` | تجاوز ربط عمود العنوان/المؤلف يدوياً. |
| `--betaka` | تفكيك عمود العنوان كبطاقة الشاملة. |
| `--canonical-split` | تقسيم العنوان على الفاصل `\|`. |
| `--extra-roles` | ضمّ الأدوار العلمية الإضافية إلى الملاحظة. |
| `--key-style` | `alalc` (رومنة) · `numeric` (ترقيم تسلسلي). |

### 3) مرسم تدبر لأنماط CSL

افتح `tadabbur-csl-studio.html` في المتصفّح مباشرةً. اضبط النمط، تأكّد من شارة الصحّة، ثم نزّل `tadabbur.csl`.

---

## 🔗 الربط بمندلي

1. نزّل `tadabbur.csl` من المرسم.
2. ارفعه إلى مستودع GitHub عام، ثم انسخ رابط **Raw**.
3. في **Mendeley Cite**: القائمة `⋯` ← **Citation Style** ← **Add a custom style** ← الصق الرابط.

> **تنبيه التأريخ المزدوج (هجري/ميلادي):** أسلوب وضع `original-date: 1392` في حقل Extra يقرؤه Zotero يقيناً؛ أمّا مندلي فلا يضمن تحويل أسطر Extra إلى متغيّرات CSL. الملف مصمَّم ليتدهور بأمان (يظهر الميلادي وحده دون خطأ). جرّب مرجعاً في مندلي عندك للتأكّد.

---

## 🛡️ أمان البيانات

- **القاعدة المصدر لا تُمسّ إطلاقاً** (قراءة فقط محكَمة).
- لا تُختلق بيانات: يُربط العمود فقط إن وُجد وكانت قيمته غير فارغة بعد التنظيف.
- إزالة المكرّرات لا تطال المصادر المتمايزة (عُنوان+مؤلف مختلفان يبقيان).

---

## ⚠️ تنبيه قبل الرفع إلى GitHub

ملف `smart_dictionary.db` كبير جداً (مئات الميغابايت) ويتجاوز **حدّ GitHub‏ 100MB‏ للملف الواحد**، فلا تَرفعه مباشرة وإلا فشل الـ`push`. أنشئ ملف `.gitignore` يتضمّن:

```gitignore
# قاعدة البيانات الكبيرة وملفاتها المؤقتة
smart_dictionary.db
*.db-shm
*.db-wal

# مخرجات مولّدة
*.bib
*.ris

# بايثون
__pycache__/
*.pyc

# سجلّات محلية
.remember/
```

> إن كنت مضطراً لمشاركة القاعدة، استعمل **Git LFS** أو ارفعها على استضافة ملفات خارجية وضع رابطها في README.
